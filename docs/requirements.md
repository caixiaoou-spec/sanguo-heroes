# 三国群英传 Web 复刻版 - 需求功能文档

> 版本：v2.0
> 更新日期：2026-03-28

---

## 1. 项目概述

### 1.1 项目简介

三国群英传 Web 复刻版是一款基于浏览器的三国题材策略 RPG 游戏，致敬经典《三国群英传》系列。游戏包含回合制战略层（世界地图）和实时战术层（战斗场景），玩家扮演三国时期一方势力的君主，通过内政、外交、征战统一天下。

### 1.2 目标平台

- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 支持桌面端和移动端触屏操作
- 逻辑分辨率 1280×800，支持 HiDPI/Retina 屏幕

### 1.3 技术约束

- 纯前端实现，零外部依赖（Vanilla JS + HTML5 Canvas + CSS）
- 无需构建工具，通过 HTTP 服务器直接运行
- 所有美术资源程序化生成（含可选本地图片加载）
- 所有音频通过 Web Audio API 程序化合成

### 2.3 世界地图（战略层）

#### 2.3.1 地图展示

- 固定视角全景地图，55 座城池以城墙图标显示，颜色对应所属势力
- 程序化地形：山脉、河流（黄河/长江，带流动动画）、区域标注；已加入地形系统（平原、山地、沼泽、沙漠、湖泊），不同地形影响行军速度
- 城池间以古地图风格虚线道路连接，表示相邻关系
- 装饰性动画旗帜（随风飘动）和飘浮云朵
- 势力领地光晕、鼠标悬停/选中高亮效果

#### 2.3.2 HUD 信息栏

| 位置 | 显示内容 |
|------|----------|
| 顶部栏 | 势力名称、金钱、粮草、声望、当前回合数、剩余行动点 |
| 底部栏 | 武将列表、外交、存档、结束回合 按钮 |

#### 2.3.3 城池管理

点击己方城池显示右侧面板：

| 操作 | 消耗 | 效果 | 条件 |
|------|------|------|------|
| 开发农业 | 1 行动点 + 1000 金 | 农业值 +（5 + 政治/10） | 城中有武将 |
| 开发商业 | 1 行动点 + 1000 金 | 商业值 +（5 + 政治/10） | 城中有武将 |
| 征兵 | 1 行动点 + 2000 金 | 获得城市人口 2% 的兵力（上限~1500） | 士气 -3 |
| 搜索武将 | 1 行动点 | 魅力相关概率发现在野武将并招募 | 城中有武将 |
| 修筑城防 | 1 行动点 + 1500 金 | 城防 +（3 + 统率/15） | 城中有武将 |
| 分配兵力 | 免费 | 将城池驻军分配给武将（上限 = 统率×50） | - |
| 调遣武将 | 1 行动点 | 将武将从当前城池移动到相邻己方城池 | 相邻有己方城池且城中有武将 |

城池面板还显示：城池名、所属势力、人口、农业、商业、城防、驻军、士气，以及驻扎武将列表（含头像）。

#### 2.3.4 武将系统

**武将属性（283 名武将）：**

| 属性 | 说明 | 范围 |
|------|------|------|
| 武力(war) | 影响物理攻击、决斗胜率 | 1-100 |
| 智力(int) | 影响策略技能威力 | 1-100 |
| 统率(lead) | 影响带兵上限、HP加成 | 1-100 |
| 政治(pol) | 影响内政效果 | 1-100 |
| 魅力(cha) | 影响招募成功率 | 1-100 |
| 等级(level) | 初始 8-18，上限 50 | 8-50 |
| 忠诚(loyalty) | 影响被策反概率 | 0-100 |
| 兵种(unitType) | 骑兵/步兵/枪兵/弓兵 | 4 种 |

**武将详情面板显示：**
- 128×128 头像（优先本地图片，回退程序化像素画）
- 等级、经验值进度条
- 五维属性数值
- HP/MP 数值条
- 已习得技能列表
- 装备栏位

**武将列表面板：**
- 表格视图展示全部己方武将
- 支持排序

#### 2.3.5 出征系统

| 步骤 | 操作 |
|------|------|
| 1. 选择目标 | 点击敌方或中立城池 |
| 2. 选择出发城 | 必须与目标相邻 |
| 3. 选择武将 | 从出发城中选择武将出战 |
| 4. 发起进攻 | 检查条件后进入战斗场景 |

**约束条件：**
- 出发城必须与目标城池相邻
- 被选武将必须有士兵
- 不能攻击盟友城池

#### 2.3.6 调遣系统

将武将从当前城池移动到相邻的己方城池，用于加强防御或集结兵力。

| 步骤 | 操作 |
|------|------|
| 1. 打开调遣面板 | 点击己方城池面板中的"调遣"按钮 |
| 2. 选择目标城池 | 从相邻己方城池中选择目标 |
| 3. 选择武将 | 勾选要调遣的武将（不限人数） |
| 4. 确认调遣 | 武将连同所带兵力移动到目标城池 |

**约束条件：**
- 目标城池必须与出发城池相邻
- 目标城池必须属于己方
- 城中至少有一名武将
- 消耗 1 行动点

#### 2.3.7 外交系统

| 操作 | 消耗 | 效果 | 成功条件 |
|------|------|------|----------|
| 结盟 | 免费 | 双方互为盟友，不可互相攻击 | 声望相关概率 |
| 宣战 | 免费 | 双方互为敌对，解除同盟 | 必定成功 |
| 停战 | 5000 金 | 双方解除敌对状态 | 声望相关概率 |
| 劝降 | 免费 | 目标势力全部城池和武将归入 | 仅对 ≤2 城势力有效，声望差相关概率 |

#### 2.3.8 回合流程

```
玩家操作阶段（3 行动点）
    ↓
点击"结束回合"
    ↓
经济结算（收入、消耗、人口增长、士气变化）
    ↓
事件处理（30% 概率触发随机事件 / 检查剧情事件）
    ↓
AI 势力行动（内政、征兵、进攻）
    ↓
AI 攻击玩家城池 → 弹出防守战斗场景
    ↓
胜负检查 → 下一回合
```

#### 2.3.9 经济系统

| 项目 | 公式 |
|------|------|
| 金钱收入 | 商业值 × 人口 / 10000 |
| 粮草收入 | 农业值 × 人口 / 3000 |
| 粮草消耗 | （驻军 + 武将带兵）× 0.1 |
| 人口增长 | 士气 > 50 时缓慢增长 |
| 士气影响 | 缺粮导致士气下降 |

#### 2.3.10 AI 系统

AI 每回合自动执行：
1. 城池开发（农业/商业）
2. 征兵补充兵力
3. 分配兵力给武将
4. 评估攻防兵力比后决定是否进攻

**AI 进攻策略：**

| 兵力比（攻/守） | 进攻概率 |
|------------------|----------|
| > 2.0 | 50% |
| > 1.5 | 30% |
| > 1.2 | 15% |
| ≤ 1.2 | 3% |

- AI 攻击 AI 城池：自动结算
- AI 攻击玩家城池：弹出战斗场景，玩家操作防守方

### 2.4 战斗系统（战术层）

#### 2.4.1 战场设定

- 1600px 宽横向卷轴战场
- 程序化地形：草地、树木、云朵、山脉
- 攻方从左侧、守方从右侧展开
- 60 秒战斗计时器

#### 2.4.2 战斗单位

每个武将统领一个小队（1 名武将精灵 + 8 名小兵精灵），具备：
- HP / MP 数值条
- 士兵数量
- 兵种特性和移动速度
- 状态机：前进 → 战斗 → 技能 → 撤退 → 阵亡
- 独立的攻击和移动动画

#### 2.4.3 兵种克制

| 兵种 | 克制对象 | 倍率 |
|------|----------|------|
| 骑兵 | 弓兵 | 1.3x |
| 枪兵 | 骑兵 | 1.4x |
| 弓兵 | 枪兵 | 1.2x |
| 步兵 | 弓兵 | 1.2x |

**攻击范围：**
- 弓兵：200 像素（远程）
- 其他：60 像素（近战）

**移动速度：**
- 骑兵：120
- 其他：80

#### 2.4.5 一骑讨（单挑）系统

- 战斗中随机触发武将单挑
- 最多 10 回合，每回合基于武力值 + 随机数比较
- 落败方武将受到大量伤害

#### 2.4.6 战斗操作

| 操作 | 方式 |
|------|------|
| 选中武将 | 鼠标点击武将精灵 |
| 释放技能 | 点击技能栏按钮 |
| 自动战斗 | 点击自动按钮切换 |
| 加速 | 点击速度按钮（1x/2x/3x） |

#### 2.4.7 战斗结算

| 结果 | 效果 |
|------|------|
| 攻方胜利 | 城池归属变更、缴获金钱粮草、俘获武将（40%+声望概率） |
| 守方胜利 | 攻方武将撤退至其他城池或被俘 |
| 超时 | 比较双方总 HP，多者获胜 |
| 经验获得 | 胜方武将获得 50 经验，败方获得 20 经验 |

#### 2.4.8 战斗 HUD

- 选中武将头像、名称、HP/MP条、士兵数
- 技能栏（含冷却遮罩）
- 双方军队标签："我军(攻/守)" vs "敌军(攻/守)"
- 60 秒倒计时
- 浮动伤害数字
- 技能视觉特效（爆炸、光束、旋转、斩击等）
- 命中粒子效果、强力技能屏幕震动

#### 2.4.9 防守战斗

当 AI 攻击玩家城池时：
- 弹出"敌军来袭"警告面板（红色主题），显示：
  - 进攻方势力名称
  - 被攻击的城池名称
  - 来犯武将列表（头像、姓名、属性、兵力、兵种）
  - 我方守军信息（武将数、总兵力、城防值）
- 玩家点击"迎战！"按钮后进入战斗场景
- 玩家控制守方武将，镜头初始位置在守方一侧
- 胜负判定适配守方视角
- 多场防守战斗按队列依次进行，每场战斗前都会弹出警告面板
